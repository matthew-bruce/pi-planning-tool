# PI Planning Tool

This is a Next.js project for PI Planning tool that includes various screens and functionalities for efficient planning.

## Project Structure

```
pi-planning-tool/
├── components/
├── pages/
│   ├── index.js
│   ├── sorting-frame.js
│   ├── team-planning.js
│   ├── dependencies.js
│   ├── dashboard.js
│   ├── triage.js
│   └── help.js
├── models/
├── providers/
├── public/
└── styles/
```

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Features
- Navigation between different screens
- Drag-and-drop functionality
- React Flow integrations for visual management
- Local storage persistence for user data
- Demo mode for simulation purposes
